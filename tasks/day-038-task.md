# Day 38 - Pull Docker Image

## Task Description

The Nautilus development team needs to prepare a container image for testing in a containerized environment on the Stratos DC infrastructure.

---

## Requirements

### 1. Image Preparation

* On **App Server 2**, pull the image:

```
busybox:musl
```

* Create a new tag for this image as:

```
busybox:news
```

---

## Expected Result

* The image `busybox:musl` is available on App Server 2.
* A new tag `busybox:news` exists for the same image.
