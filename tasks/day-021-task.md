# Day 21 - Set Up Git Repository on Storage Server

## Task Description

The Nautilus development team needs a new Git repository to support an upcoming application project on the Storage Server.

---

## Requirements

### 1. Package Installation

* Install **Git** using:

```
yum
```

---

### 2. Repository Creation

* Create a **bare** repository with the exact name:

```
/opt/apps.git
```

---

## Expected Result

* Git is installed on the server.
* The directory `/opt/apps.git` exists and is initialized as a bare repository.

---

## Notes

* Do not create a working tree inside this directory.
* Ensure the repository is bare.

---

End of Task
