# Day 36 - Deploy Nginx Container on Application Server

## Task Description

The Nautilus DevOps team is validating container-based application deployments on the Stratos DC infrastructure.

---

## Requirements

### 1. Container Deployment

* On **Application Server 2**, create a container with:

```
Name: nginx_2
Image: nginx:alpine
```

* Ensure the container is running.

---

## Expected Result

* The container `nginx_2` exists and is in a running state.
* The nginx service is active inside the container.
