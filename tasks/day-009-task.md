# Day 9 - MariaDB Troubleshooting

## Task Description

The Nautilus application is currently unable to connect to its database due to a MariaDB service failure on the Database Server.

---

## Requirements

### 1. Issue Investigation

* Identify why the **MariaDB** service is not running on the DB Server.

---

### 2. Service Restoration

* Start and ensure the MariaDB service is operational.

---

### 3. Validation

* Confirm the application can successfully connect to the database.

---

## Expected Result

* MariaDB is running without errors.
* Database connectivity is restored for the Nautilus application.

---

End of Task
