# Day 69 - Install Jenkins Plugins

## Task Description

Install required CI/CD plugins on the Jenkins server to support Git-based workflows and GitLab integrations.

## Requirements

### Access Jenkins UI

* Open Jenkins using the top navigation button.
* Login with:

```
Username: admin
Password: Adm!n321
```

---

### Install Required Plugins

* Navigate to the plugin management section (Update Center).
* Install the following plugins:

```
Git
GitLab
```

* If prompted, select the option to:

```
Restart Jenkins when installation is complete and no jobs are running
```

* Wait for Jenkins to fully restart before attempting to log in again.

---

### Documentation

* Capture screenshots or record the process for review if required.

## Expected Result

* Git and GitLab plugins are successfully installed.
* Jenkins restarts (if required) without errors.
* Jenkins UI becomes accessible again after restart.
* Plugins appear as installed in the plugin management section.

End of Task
