# Day 5 - SElinux Installation and Configuration

## Task Description

The xFusionCorp security team is preparing App Server 3 for SELinux-based security testing in the Stratos DC environment.

---

## Requirements

### 1. Package Installation

* Install the required **SELinux** packages on **App Server 3**.

---

### 2. SELinux Mode

* Configure SELinux to be **permanently disabled**.

---

### 3. Reboot Policy

* Do **not** reboot the server manually.
* The change must take effect after the scheduled reboot.

---

## Expected Result

* SELinux is disabled after the next system reboot.
* Required SELinux packages are present on the system.

---

End of Task
