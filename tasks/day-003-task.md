# Day 3 - Secure Root SSH Access

## Task Description

The xFusionCorp security team requires improved access control by disabling direct root login over SSH on all App Servers in the Stratos DC environment.

---

## Requirements

### 1. SSH Configuration

* Disable **direct root SSH login** on **all App Servers**.

---

### 2. Access Policy

* Root access must be restricted to non-SSH methods (e.g., sudo).

---

## Expected Result

* SSH login as `root` is not permitted on any App Server.
* Servers remain accessible through standard user accounts.

---

End of Task
