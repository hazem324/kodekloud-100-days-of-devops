# Day 31 - Git Stash

## Task Description

The Nautilus development team needs to recover previously stashed work and synchronize it with the central repository.

---

## Requirements

### 1. Repository Scope

* Local path:

```
/usr/src/kodekloudrepos/blog
```

---

### 2. Stash Recovery

* Locate the existing stashes.
* Apply the stash with identifier:

```
stash@{1}
```

---

### 3. Version Control

* Commit the restored changes.
* Push the commit to the remote repository.

---

## Expected Result

* The changes from `stash@{1}` are restored successfully.
* The update is committed and available in the remote origin.

---

End of Task
