# Day 4 - Script Execution Permissions

## Task Description

The xFusionCorp sysadmin team needs to correct execution permissions for a backup automation script on App Server 2.

---

## Requirements

### 1. Script Location

* Target script:

```
/tmp/xfusioncorp.sh
```

---

### 2. Permission Update

* Grant executable permission to the script.
* Ensure **all users** are allowed to execute it.

---

## Expected Result

* The script `/tmp/xfusioncorp.sh` is executable.
* Any system user can run the script without permission errors.

---

End of Task
