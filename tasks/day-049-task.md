# Day 49 - Deploy Applications with Kubernetes Deployments

## Task Description

Create a Kubernetes Deployment to manage an Nginx application using the specified container image.

## Requirements

* Create a deployment with the following name:

```
nginx
```

* Use the container image:

```
nginx:latest
```

* Explicitly specify the image tag.
* Perform the operation using the `kubectl` utility configured on `jump_host`.

## Expected Result

* A deployment named `nginx` exists in the cluster.
* The deployment uses the `nginx:latest` image.
* Managed pods are created and running under the deployment.

End of Task
