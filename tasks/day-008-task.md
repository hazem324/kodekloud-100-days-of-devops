# Day 8 - Install Ansible

## Task Description

The Nautilus DevOps team is initiating configuration management automation using Ansible, with the Jump Host acting as the Ansible controller.

---

## Requirements

### 1. Installation Method

* Install **Ansible version 4.8.0** using:

```
pip3
```

only.

---

### 2. System Scope

* Ensure the Ansible binary is available **globally**.
* All system users must be able to execute Ansible commands.

---

## Expected Result

* Ansible 4.8.0 is installed on the Jump Host.
* The `ansible` command is accessible for all users.

---

End of Task
